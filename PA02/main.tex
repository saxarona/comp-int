{\large \textbf{Multi-layer perceptron}}

\begin{itemize}
  \item A two-layer perceptron was implemented.
  The perceptron correctly classifies a non-linear model.
  \item The activation function used was the \textbf{hard limit}, which maps a value $n$ to 0 if negative, or 1 otherwise.
  \item The code was implemented in Octave, using the codes provided.
  \item A \texttt{readme.txt} file is included with the details needed for execution, along with the Texinfo per-module documentation, i.e. you can use \texttt{help multiperceptron} in the Octave prompt for instructions.
\end{itemize}
